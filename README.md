# Angular Burger App

# Использованые библиотеки в приложении
- Angular 
- TypeScript
- CSS3/HTML5
- API запросы с testologia.site
- Angular/forms

## Development server

Run `npm install` or `yarn install`

Run `ng serve` for a dev server. 

Navigate to `http://localhost:4200/`
